#### To run the code
- First, create an output folder:
```
mkdir out
```
- Next, compile the program:
```
javac *.java -d out
``` 
- Run the program afterwards:
```
cd out
java RunqueueTester <option>
```
- There are 3 options: "tree", "array", "linkedlist"
